rWifi Privacy Policy

rWifi (Rust TUI Wi-Fi Station Manager) is built with a commitment to user privacy. As a local terminal application, rWifi does not track, collect, or transmit your personal data.

Data Collection and Telemetry
rWifi does not include any analytics, crash reporting platforms, or tracking metrics.
rWifi does not collect or request any personal identifiers, email addresses, or system credentials.

Network Connections
rWifi runs completely offline and does not make any network connections. The Wi-Fi manager only interfaces with the local system's physical network adapters using native Windows APIs.

Local Data Storage
All data managed by rWifi is kept strictly on your local machine under your user profile directory:
Configurations: Stored at %APPDATA%\rWifi\config.yaml
Log Files: Diagnostics are logged locally to %APPDATA%\rWifi\log.txt

rWifi does not upload your local configuration files or logs to any remote server.
